Tentang Sistem

SmartTraffic AI adalah sistem pemantauan lalu lintas cerdas yang menggunakan teknologi Computer Vision (YOLOv8) untuk mendeteksi dan menghitung kendaraan secara real-time. Sistem ini tidak hanya menghitung volume lalu lintas, tetapi juga menganalisis tingkat kepadatan untuk memberikan rekomendasi rute alternatif tercepat kepada pengguna.

1. Deteksi

Kamera CCTV mengambil gambar jalan raya. AI mendeteksi objek mobil dan motor setiap detik.

2. Analisis

Sistem menghitung kepadatan dan menentukan status (Lancar/Padat/Macet) serta menghitung estimasi delay.

3. Rekomendasi

Dashboard menampilkan peta interaktif dan memberikan saran rute tercepat menghindari titik macet.

Fitur Unggulan

Klasifikasi Kendaraan

Membedakan antara Mobil dan Sepeda Motor dengan akurasi tinggi menggunakan model Deep Learning.

Smart Routing

Algoritma pencari jalan yang mempertimbangkan kemacetan real-time untuk memberikan rute alternatif.

Statistik & History

Grafik volume lalu lintas per jam/hari dan kemampuan ekspor data ke format CSV.

Multi-Camera

Mendukung pemantauan banyak titik kamera sekaligus dalam satu dashboard terpusat.

Referensi API

GET /api/stats

Mengambil data statistik lalu lintas real-time (jumlah kendaraan, status per kamera).

GET /metrics

Endpoint untuk scraping Prometheus (monitoring infrastruktur).

GET /export/csv

Mengunduh laporan data lalu lintas dalam format file CSV.
